infinite-carousel-wc

An infinite carousel Web Component built with modern CSS and just a little Javascript

This small, simple web component has no dependencies and can be consumed by vanilla JS or any front-end framework. It is implemented using modern CSS and Javascript standards for maximum performance.

  • Lightweight (~1.5kb gzipped)
  • No Dependencies
  • CSS re-ordering instead of JS DOM manipulation
  • Horizontal or Vertical
  • Touch-friendly
  • Minimal Javascript
